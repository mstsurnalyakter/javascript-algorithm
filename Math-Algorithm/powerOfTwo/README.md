 # Power of two

 An integer is a `power of two` if the there exists an integer `x` such that `n` `===` `2^x`

 ## Examples:

 ```
Input : n = 4
Output : Yes
Explanation: 2^2 = 4

Input : n = 32
Output : Yes
Explanation: 2^5 = 32
 ```
